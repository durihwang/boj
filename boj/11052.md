## 문제
https://www.acmicpc.net/problem/11052

## 알고리즘
다이나믹 프로그래밍

## 풀이
### 점화식
``` d[i] = max(d[i-j]) + p[j]```

- 1번째 카드팩은 1개의 카드
- 2번쨰 카드팩은 2개의 카드
- n번째 카드팩은 n개의 카드
- 위 중에서 카드 가격이 가장 높은 것을 골라야 한다.
- 이 문제는 한번 계산된 d[i]가 유지되는 것이 아니라 1번째 카드부터 현재 나의 카드팩까지의 경우의 수를 계속 계산해서 갱신해 주어야 한다.


### 예시
n = 4

카드가격

1 5 6 7

- 1번째 카드팩의 최대 가격은 1 -> 하나밖에 못고름
- 2번째 카드팩의 최대 가격은 5 -> 1번쨰 카드 두개 가격은 2, 2번째 카드 하나의 가격은 5
- 3번째 카드팩의 최대 가격은 6 -> 3번쨰 카드 한개 가격은 6, 2번쨰 카드 둘과 1번째 카드 하나 가격은 1, 1번째 카드 3개 가격은 3
- 위 예시처럼 내 현재 카드팩이 3번쨰면 1번쨰부터 3번째까지 계속 가격을 갱신해 주어야 한다.



## 코드
```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.lang.reflect.Array;
import java.util.*;

class Main {

    static int n;
    static int k;
    static int answer;
    static boolean check;
    static int[] d;
    static int[] p;
    static ArrayList<Long> reduction = new ArrayList<>();

    public static void main(String[] args) throws IOException {

        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        n = Integer.parseInt(br.readLine());
        d = new int[n + 1];
        p = new int[n + 1];
        StringTokenizer st = new StringTokenizer(br.readLine());
        for (int i = 1; i <= n; i++) {
            p[i] = Integer.parseInt(st.nextToken());
        }

        d[0] = 0;

        // 1부터 n번째 까지의 카드팩의 가격 모두 구한다.
        for (int i = 1; i <= n; i++) {

            // 1은 1번, 2는 2번 ... n은 n번 카드를 얻을 수 있다.
            for (int j = 1; j <= i; j++) {
                d[i] = Math.max(d[i], d[i - j] + p[j]);
            }
        }

        System.out.println(d[n]);

    }

}
```
